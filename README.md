# Patient Vital Status Checker

This is a PHP mini project to validate patient vitals like:
- Temperature
- Pulse
- Blood Pressure

## Concepts Used
- Arrays
- Functions
- Higher Order Function
- Callback
- Recursion
  

## How to Run
1. Install  WAMP
2. Place project inside www
3. Run index.php in browser
